Advisor-style XAS and EXAFS reference dashboard

The selected reference is shown as three stacked panels: pre-edge or XANES zoom, normalized XANES edge, and FT-EXAFS magnitude. Data are loaded from external JSON files generated from Athena/Larch outputs.

Data-loading model: This dashboard expects dashboard_export/database_index.json and sample-level JSON files for XANES, pre-edge, FT-EXAFS, and metadata. It is GitHub Pages-ready if the folder structure is preserved.
